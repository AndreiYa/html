Теги dl dt dd

Описание

Тег <dl> входит в тройку элементов <dl>, <dt>, <dd>, предназначенных для создания списка определений.
Каждый такой список начинается с контейнера <dl>, куда входит тег <dt> создающий термин и тег <dd> задающий определение этого термина.

Пример использования

<dl>
    <dt>Что такое FAQ?
    <dd>FAQ (англ. frequently asked questions) — страница с часто задаваемыми вопросами по какой-либо теме с ответами на них.
</dl>

Результат

Что такое FAQ?
FAQ (англ. frequently asked questions) — страница с часто задаваемыми вопросами по какой-либо теме с ответами на них.

Тег pre

Описание

Элемент <pre> определяет блок предварительно форматированного текста. Такой текст отображается обычно моноширинным шрифтом и со всеми пробелами между словами.
По умолчанию, любое количество пробелов идущих в коде подряд, на веб-странице показывается как один.
Тег <pre> позволяет обойти эту особенность и отображать текст как требуется разработчику.

Пример использования

<body>
    <pre>
        Тег <pre> сохранит форматирование текста с точным количеством
        проставленных пробелов и переносов строк.
            как в исходном коде,     
        К тому же, текст будет моноширинным.
        Моноширинный – все буквы одинаковой ширины (машинописный шрифт).
    </pre>
</body>

Тег tt

Описание

Тег <tt> отображает текст моноширинным текстом. Этот тег относится к группе тегов физического форматирования.

Пример использования

<body>
    <p>Это шрифт вашего браузера по умолчанию.</p>
    <p><tt>А это моноширинный шрифт.</tt></p>
</body>

Результат

Hello World Это шрифт вашего браузера по умолчанию.

Hello World А это моноширинный шрифт.

Тег code

Описание

Тег <code>предназначен для отображения одной или нескольких строк текста, который представляет собой программный код.
Сюда относятся имена переменных, ключевые слова, тексты функции и т.д.
Браузеры обычно отображают содержимое контейнера <code> как моноширинный текст уменьшенного размера.
В отличие от тега <pre> дополнительные пробелы внутри контейнера <code> не учитываются, так же, как и переносы текста. Поэтому используйте тег <br> или <p> для создания переносов.

Пример использования

<body>
    <p><code>
            function checkParent (src, dest) {<Br>
             while (src != null) {<Br>
               if (src.tagName == dest) return src<Br>
                src = src.parentElement<Br>
             }<Br>
             return null<Br>
            }
    </code></p>
</body>

Результат

function checkParent (src, dest) {
while (src != null) {
if (src.tagName == dest) return src
src = src.parentElement
}
return null
}

Тег kbd

Описание

Тег <kbd>используется для обозначения текста, который набирается на клавиатуре или для названия клавиш.
Браузеры обычно помечают текст в контейнере <kbd> моноширинным шрифтом.

Пример использования

<body>
    <p>
        Для закрашивания выделенной области фоновым цветом воспользуйтесь комбинацией 
        <kbd><Ctrl>+<Backspace></kbd> или <kbd><Ctrl>+<Delete></kbd>.
    </p>
</body>

Результат

Для закрашивания выделенной области фоновым цветом воспользуйтесь комбинацией <Ctrl>+<Backspace> или <Ctrl>+<Delete>.

Тег var

Описание

Тег <var>используется для выделения переменных компьютерных программ. Браузеры обычно помечают текст в контейнере <var> курсивным начертанием.

Пример использования

<body>
    <p>
        В данном примере делается запрос к базе данных для получения поля <var>content_title</var> из таблицы <var>content</var>.
    </p>
</body>

Результат

В данном примере делается запрос к базе данных для получения поля content_title из таблицы content.

Теги sub и sup

Описание

Тег <sup> Отображает шрифт в виде верхнего индекса. Шрифт при этом отображается выше базовой линии текста и уменьшенного размера.

Тег <sub> Отображает шрифт в виде нижнего индекса. Шрифт при этом отображается ниже базовой линии текста и уменьшенного размера.

Пример использования

Для <sup>

<body>
    <p>
        c<sup>2</sup>=a<sup>2</sup>+b<sup>2</sup>
    </p>
</body>

Для <sub>

<body>
    <p>
        C<sub>2</sub>H<sub>5</sub>OH
    </p>
</body>

Результат

Для <sup>

c2=a2+b2

Для <sub>

C2H5OH

Теги blockquote и q

Описание

Тег <blockquote>предназначен для выделения длинных цитат внутри документа. Текст, обозначенный этим тегом, традиционно отображается как выровненный блок с отступами слева и справа (примерно по 40 пикселов), а также с отбивкой сверху и снизу.

Тег <q>используется для выделения в тексте цитат. Содержимое контейнера автоматически отображается в браузере в кавычках.

Пример использования

Для <blockquote>

<body>
    <blockquote>
        <p>
            В свое  время  драматург  Михаил  Львовский  написал  песню,
            которую студенчество по праву считает  "своей",  хотя  написана
            она была не студентом и не о студентах. Это песня  "Глобус".  В
            ней есть слова, которые многое объясняют: "Потому что нам
            нельзя без песен, Потому что  мир  без  песен тесен!"
        </p>
    </blockquote>
</body>

Для <q>

<body>
    <p>
    Станислав Лец утверждал:<q>Чаще всего выход там, где был 
        вход</q>.
    </p>
</body>

Результат

Для <blockquote>

В свое время драматург Михаил Львовский написал песню,
которую студенчество по праву считает "своей", хотя написана
она была не студентом и не о студентах. Это песня "Глобус". В
ней есть слова, которые многое объясняют: "Потому что нам
нельзя без песен, Потому что мир без песен тесен!"

Для <q>

Станислав Лец утверждал: Чаще всего выход там, где был вход.

Теги ins и del

Описание

Тег <ins>предназначен для выделения текста, который был добавлен в новую версию документа. Подобное форматирование позволяет отследить, какие изменения в тексте документа были сделаны. Браузеры обычно помечают текст в контейнере <ins> как подчеркнутый.

Тег <del>используется для выделения текста, который был удален в новой версии документа. Подобное форматирование позволяет отследить, какие изменения в тексте документа были сделаны. Браузеры обычно помечают текст в контейнере <del> как перечеркнутый.

Пример использования

<body>
    <p>
        Заметим, что схожий результат с Firefox показывает и браузер 
        Opera, но, как видно на рис. 1, в последующих версиях <del>этот 
        недостаток был устранен</del> <ins>подход разработчиков этого 
        браузера был изменен</ins>.
    </p>
</body>

Результат

Заметим, что схожий результат с Firefox показывает и браузер
Opera, но, как видно на рис. 1, в последующих версиях этот
недостаток был устранен подход разработчиков этого
браузера был изменен.

Тег address

Описание

Тег <address>предназначен для хранения информации об авторе и может включать в себя любые элементы HTML вроде ссылок, текста, выделений и т.д. Планируется, что поисковые системы будут анализировать содержимое этого тега для сбора информации об авторах сайтов.

Пример использования

<body>
    <address>Kuprevicha, 1/1, 220141, Minsk, Belarus. Phone: +375.17.2686911</address>
</body>

Результат

Kuprevicha, 1/1, 220141, Minsk, Belarus. Phone: +375.17.2686911

Теги nobr и wbr

Описание

Тег <nobr>уведомляет браузер отображать текст без переносов.

Тег <wbr>указывает браузеру место, где допускается делать перенос строки в тексте, если этого требует ширина родительского элемента.

Пример использования

Для <nobl>

<body>
        <p>
            <nobl>В свое  время  драматург  Михаил  Львовский  написал  песню,
            которую студенчество по праву считает  "своей",  хотя  написана
            она была не студентом и не о студентах. Это песня  "Глобус".  В
            ней есть слова, которые многое объясняют: "Потому что нам
            нельзя без песен, Потому что  мир  без  песен тесен!"</nobl>
        </p>
</body>

Для <wbr>

<body>
    <p>
        метоксихлор<wbr>диэтиламино<wbr>метил<wbr>бутил<wbr>амино<wbr>акридин
    </p>
</body>

Результат

Для <nobl>

Может привести к появлению горизонтальной полосы прокрутки.

Для <wbr>

При изменении ширины окна текст будет переноситься в тех местах, где указан тег <wbr>.
